# Synthetic Army Housing Utilization & Vacancy Dashboard

## Overview
This project analyzes fully synthetic Army housing data to identify occupancy trends, vacancy risks, low-utilization buildings, and data quality issues.

## Business Problem
Senior leaders need a repeatable reporting process to monitor housing utilization, identify buildings below target occupancy, and support resource planning decisions.

## Dataset
The dataset is fully synthetic. It does not contain real government, military, personal, or operational data.

## Tools Demonstrated
- Excel-ready CSV data
- SQL analysis
- Data dictionary documentation
- Data quality checks
- Dashboard design planning
- Executive-ready findings and recommendations

## Suggested Dashboard Visuals
1. KPI cards: Total Beds, Occupied Beds, Vacant Beds, Average Net Use %
2. Bar chart: Net Use % by Installation
3. Stacked bar chart: Occupied vs Vacant Beds by Installation
4. Line chart: Net Use % by Month
5. Heatmap: Housing Type by Installation
6. Table: Buildings Below Utilization Target

## Key Business Questions
1. Which installations have the highest vacancy?
2. Which buildings are below utilization targets?
3. Which housing types have the greatest unused capacity?
4. Are data quality issues affecting reporting accuracy?

## Portfolio Summary
This project shows how raw operational data can be cleaned, validated, analyzed, and turned into leadership-ready insights.
