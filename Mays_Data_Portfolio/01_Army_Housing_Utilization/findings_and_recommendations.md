# Findings and Recommendations

## Key Findings
1. Vacancy is concentrated across a smaller group of installations and building types.
2. Buildings with lower condition scores tend to show weaker utilization.
3. Training housing shows seasonal utilization movement.
4. Raw data contained issues such as missing building numbers, invalid occupancy values, duplicate records, and inconsistent installation names.

## Recommendations
1. Create a monthly utilization exception report for buildings below 70% utilization.
2. Standardize installation names, building numbers, and design category codes before dashboard refresh.
3. Track high-vacancy buildings alongside condition score and open work orders.
4. Use recurring SQL data quality checks before sending leadership reports.
