# One-Page Executive Summary

## Purpose
This synthetic analysis reviews housing utilization, vacancy, condition score, work orders, and data quality issues across fictional installations.

## Bottom Line
The analysis shows how recurring housing data can be cleaned, analyzed, and converted into leadership-ready insights. Several fictional buildings fall below utilization targets and should be reviewed alongside condition score and open work order volume.

## Key Findings
1. Vacancy is concentrated in a small number of installations and housing types.
2. Lower condition scores are associated with weaker utilization.
3. Raw data quality issues can affect reporting accuracy.

## Recommendations
1. Create a monthly utilization exception report.
2. Standardize installation names, building numbers, and design category codes.
3. Track high-vacancy buildings alongside condition score and open work orders.
4. Run SQL data quality checks before dashboard refresh.

## Data Disclaimer
All data in this portfolio project is synthetic. No real government, military, operational, or personal data is included.
