# Anthony Mays Data Portfolio Starter Package

## Portfolio Theme
Data analysis, BI reporting, data quality, SQL, dashboards, and executive-ready decision support.

## Public Portfolio Summary
This portfolio highlights synthetic data projects focused on operational reporting, data quality, dashboard development, SQL analysis, and leadership-ready insights. All datasets are synthetic and safe for public use.

## Included Projects
1. Synthetic Army Housing Utilization & Vacancy Dashboard
2. Data Quality Validation Framework for Synthetic Housing Data
3. Synthetic HR Hiring Pipeline & Workforce Reporting Dashboard
4. CRM Service Desk Performance & Backlog Dashboard
5. Executive Housing Readiness Briefing Using Synthetic Data

## How to Use
1. Upload this folder to GitHub.
2. Upload the CSV files to Tableau Public or Power BI.
3. Build dashboards using the suggested visuals in each README.
4. Add your best dashboard links to LinkedIn Featured.
5. Add your portfolio link to your resume.

## Important Disclaimer
All data in this package is synthetic. It does not include real government, military, personal, applicant, employee, or operational data.
